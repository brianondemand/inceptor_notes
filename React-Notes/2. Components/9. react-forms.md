## Understanding Forms in React

Forms in React can be a bit different than traditional HTML forms. Instead of relying solely on the DOM, we control form inputs with Reactâ€™s state.

1. **Controlled Components:** A controlled component is a component that does not maintain its own state but instead uses React state. The inputâ€™s value is determined by the state.
